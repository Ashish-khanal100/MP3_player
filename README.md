# MP3 Player

A simple MP3 Player web application built using Node.js, HTML, CSS, and JavaScript.
This application allows users to play songs directly from the songs folder with features like play, pause, volume control, and audio progress tracking.

---

## Features

* Play songs
* Pause songs
* Adjust volume
* Audio progress control
* Dynamic song loading from folder

---

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js

---

---

## Installation

Install required dependencies:

```bash
npm install
```

---

## Run the Project

Start the server:

```bash
node server.js
```

Then open your browser and visit:

```bash
http://localhost:3000
```

---

## Important Note

The `node_modules` folder is not uploaded to GitHub because it can be recreated using:

```bash
npm install
```
